Манифест проекта

1. Имя для мабы данных:
    CREATE DATABASE shop;

2. Скрипты для создания структуры базы данных разбивать на файлы,
так чтобы каждый файл содержал только одну таблицу.

3. Новые скрипты с таблицами записывать следующим образом:
    Table*Name*.sql

4. Для запуска ER диаграмы в MySQL Workbench:
    Открыть базу данных в MySQL Workbench. Сверху в окне найти Database
    Database -> Reverse Engineer
    Кликать даллее и не забыть выбрать базу shop галочкой.
    После завершения процесса нужно сохранить файл в любом удобном месте.
