Манифест проекта

1. Имя для мабы данных:
    CREATE DATABASE shop;

2. Скрипты для создания структуры базы данных разбивать на файлы,
так чтобы каждый файл содержал только одну таблицу.
Создвние общего файла должно быть автоматическим.

3. Новые скрипты с таблицами записывать следующим образом:
    Table*Name*.sql - где *Name* - имя таблицы в CamelCase, во множественном числе.
    Пример: TableUsers.sql

4. Скрипты для создания таблиц должны содержать комментарии
    с описанием структуры таблицы.

5. Названия таблиц и полей должны быть на английском языке.

6. Все названия кроме файлов должны быть в нижнем регистре
    с нижним подчеркиванием между словами.

7. Таблицы к созданию находятся в script.sql.

8. Для запуска ER диаграмы в MySQL Workbench:
    Открыть базу данных в MySQL Workbench. Сверху в окне найти Database
    Database -> Reverse Engineer
    Кликать даллее и не забыть выбрать базу shop галочкой.
    После завершения процесса нужно сохранить файл в любом удобном месте.

9. Очередь выполения скриптов для правильной имплементации соответсвует номерам в файле script.sql
